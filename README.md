# Career Compass

## About The Project

Career Compass is a frontend-based career assessment web application designed to help students evaluate their placement readiness and improve their career preparation.

The application analyzes user inputs such as CGPA, projects, certifications, internship experience, and technical skills to generate a placement readiness score along with personalized improvement recommendations.

It provides an interactive dashboard where users can view their performance report, track their progress, and download a placement readiness report in PDF format.

## Features

- Placement Readiness Score Calculation
- Skill-based career analysis
- Personalized improvement recommendations
- Interactive dashboard
- CGPA, projects, certifications, and internship evaluation
- Dark/Light mode toggle
- Data storage using Local Storage
- Generate and download PDF report
- Responsive and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- DOM Manipulation
- Local Storage
- jsPDF Library

## How It Works

1. User enters career-related details
2. Application calculates placement readiness score
3. Skills and profile data are analyzed
4. Personalized recommendations are generated
5. User can view and download the final report

## Project Purpose

Career Compass was built to create a practical career assessment tool while improving frontend development skills, JavaScript logic building, and user experience design.
